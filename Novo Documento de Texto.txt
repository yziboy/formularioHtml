    <p>a)Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?</p>
    <p>Resposta: O atributo "placeholder" sinaliza o tipo de informação em texto irá ser inserido</p>
    <p>b)Na tag < input > com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?</p>
    <p>Resposta: O atributo checked determina qual das "checkbox" deverá estar pré-selecionado quando a página abrir</p>
    <p>c)Na tag < input > para que serve o atributo "value"?</p>
    <p>Resposta: O atributo value dirá o que estará pré-escrito no espaço de input</p>
    <p>d)Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?</p>
    <p>Resposta: "input type"</p>
    <p>e)Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?</p>
    <p>Resposta: A tag "label" em HTML é usada para associar um rótulo descritivo a um elemento de formulário, como um campo de entrada como "input" ou um botão.</p>
    <p>f)Na tag "option" para que serve o atributo "selected"?</p>
    <p>Resposta: Na tagem HTML, o atributo "selected" serve para especificar qual opção de seleção deve ser predefinida (ou seja, selecionada) quando a lista suspensa (ou menu suspenso) correspondente for exibida pela primeira vez na página. <br>Isso permite que você defina qual opção deve ser exibida como o valor padrão quando o usuário abrir o menu suspenso.</p>
    <p>g)Nas tags de formulário para que serve o atributo "required"?</p>
    <p>Resposta: Esse é um atributo booleano usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo.</p>
